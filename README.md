# Face Recognition Attendance System (Java + OpenCV)

## 📌 Project Overview

The Face Recognition Attendance System is a Java-based application that automatically records attendance using face recognition technology.
The system detects and recognizes faces through a webcam and marks attendance in a CSV file.

This project demonstrates the use of **Computer Vision, OpenCV, and Java programming** to automate traditional attendance systems.

---

## 🚀 Features

* Face detection using OpenCV
* Face recognition for identifying students
* Automatic attendance recording
* Dataset creation for training faces
* Training model for face recognition
* Attendance stored in CSV file

---

## 🛠 Technologies Used

* Java
* OpenCV
* Computer Vision
* Machine Learning (Face Recognition)

## ⚙️ How the System Works

1. Capture images of the student using a webcam.
2. Create a dataset of facial images.
3. Train the face recognition model.
4. Detect and recognize faces in real time.
5. Automatically record attendance in a CSV file.
